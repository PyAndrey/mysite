Задача
Задача состоит в том, чтобы реализовать REST API для приложения task tracker с использованием Django и DRF.
Экземпляры:
1. Пользователь
2. Проект
3. Задача
Пользователь должен иметь возможность зарегистрироваться. 
После регистрации Пользователь может:
• Войдите в систему, используя ник и пароль, которые были использованы для регистрации                                                                  +
• Создать новый проект                                                       +
• См. список проектов                                                        +
• Создание задачи в рамках проекта                                           +
• См. Задачи в области проекта                                               +
• Поиск задачи по названию
• Получить отдельную задачу, просмотреть и отредактировать ее содержимое (название, описание)
• Изменение статуса задачи (ожидание, реализация, проверка, выпуск)
• Фильтрация задач по статусу.

Приложение "task tracker"
Юзер (Регистрация, вход) - Проект (Создать, список проектов) - Задачи (Создать, список задач)
